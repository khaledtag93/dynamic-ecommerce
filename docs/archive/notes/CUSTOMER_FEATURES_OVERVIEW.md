# Customer Features Overview

## Storefront Features
### Shopping
- Browse products and categories
- Add products to cart
- Support for product variants
- Cart quantity updates and remove item actions

### Coupon Support
- Enter coupon code in cart
- Coupon discount appears in cart and checkout summary
- Applied coupon is saved into the order snapshot

### Checkout
- Customer information form
- Shipping and optional billing address flow
- Payment method selection
- Order notes support
- Live order summary before placing the order

### Orders
- Success page after checkout
- My Orders listing page
- Order details page with:
  - status badges
  - payment status
  - order timeline
  - totals summary
  - customer and shipping info
- Customer can cancel eligible orders before completion

## Admin Features
### Orders Management
- Orders list with filters
- Order details screen
- Status update flow
- Coupon visibility on orders
- Refund recording flow
- Refund history per order

### Coupons Management
- Create coupons
- Edit coupons
- Delete coupons
- Control coupon type, amount, schedule, and usage limits

## Current Payment Structure
- Cash on Delivery supported
- Bank Transfer supported in structure
- Ready for future online payment integration
